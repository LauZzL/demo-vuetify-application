# demo-vuetify

## 介绍

这是一个Vuetify的应用程序模板，你可以在这里开始你的项目。

它包含了一些基本的配置，以及一些基本的组件。

## 技术栈

- Vue2
- Vuetify
- Vue Router

## 安装依赖

```bash
yarn install
```

## 运行

```bash
yarn serve
```

## 打包

```bash
yarn build
```
